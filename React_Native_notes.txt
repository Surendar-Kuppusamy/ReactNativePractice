React Native:
-------------

*) First setup everything for react native.[https://reactnative.dev/docs/environment-setup]
*) Install node js and Java SE development Kit(JDK)[openjdk11].
*) Download, install android and also follow the instruction on the react native setup guide.[Follow the instruction and check various properties on android]


*) Create virtual device on android.
*) Start project, by command "npm start"
*) Next command "npm run android"

 *) npm install -g eas-cli

 *) Install flipper, expo install react-native-flipper react-devtools-core
 *) npm install expo-community-flipper

 {
  "expo": {
    // ...
    "plugins": ["expo-community-flipper"]
}

// App.tsx
import { connectToDevTools } from "react-devtools-core";if (__DEV__) {
  connectToDevTools({
    host: "localhost",
    port: 8097,
  });
}
